Pottery E-commerce

This is an intricate Business to Consumer (B2C) e-commerce website for a Colombian pottery artist. This undertaking reflects my commitment to merging aesthetics with seamless functionality. Navigating through the site, users can effortlessly select pottery pieces and add them to their virtual carts, thanks to an intuitive user interface. The website extends its reach to a global audience, facilitating secure transactions with international credit cards. Implemented with precision, a dynamic discount code system enhances user engagement. Post-purchase, the process concludes with the generation of detailed receipts, ensuring a professional and transparent transaction experience. This project exemplifies not only my technical prowess but also my dedication to delivering a sophisticated and efficient web app design.
